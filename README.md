## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
Install JDK >= 11

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/audhumla/gradle-kotlin-dsl-java-project-template.git
   ```
2. Install NPM packages
   ```sh
   ./gradlew build
   ```
